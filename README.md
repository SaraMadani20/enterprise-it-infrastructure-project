# Enterprise IT Infrastructure Project

This project simulates a complete enterprise IT infrastructure deployment for a company with three departments:
- Sales
- Marketing
- HR

## Technologies Used
- Windows Server 2019
- Active Directory
- DNS
- DHCP
- Group Policy
- WSUS
- WDS
- VPN
- File Server

## Key Features
- Active Directory deployment
- Department-based policies
- Drive mapping
- VPN configuration
- WSUS & WDS setup
- DNS forwarding and website blocking
- Backup server implementation
- Printer management
- Access restrictions using GPO
